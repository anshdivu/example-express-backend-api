> This is a Github Repo Template to createNode.js/Express.js based Backend REST APIs

## Prerequisites

- node ^10.16.0
- yarn ^1.13.0

## Install

```sh
yarn install
```

## Usage

```sh
# for prod deployment
yarn build
yarn start

# OR #

# for local development
yarn start:ts

# OR #

# for local development with file watch (auto-restart)
yarn start:watch
```

## Run tests

```sh
yarn test

# OR #

yarn test:watch
```
